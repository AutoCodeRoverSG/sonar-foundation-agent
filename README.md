# Sonar Foundation Agent

Sonar Foundation Agent is a coding agent for general software issues, developed at Sonar by the
former AutoCodeRover team. As of December 19th, 2025, Sonar Foundation Agent scores 79.4% on
SWE-bench Verified and 52.62% on SWE-bench, while maintaining a low average cost of $1.98 and
a high efficiency of 10.4 minutes per issue.
